What is Responsive Design?

All finished, and ready for more? Great! Let's keep going! The web has seen numerous changes over the years. One such change is called "responsive design". If you haven't heard of it, don't worry. I am here to help. Responsive design, put simply, allows your website to be displayed in a concise manner across several different electronic devices. This means that one site can be easily accessed on PCs, tablets, and smartphones, to help you reach a broader audience. Web Accessibility

Lastly, I would like to take a moment to address web accessibility. Certain disabilities can inhibit a user's ability to navigate through your website. Low contrast colors can be difficult for those with varying types of color blindness to read the text on your site. Certain neurological disorders and injuries to the hand can render a user incapable of using a mouse; which can make it difficult to navigate between links on your site. Those with severly impared vision, or those without the ability to see rely on screen readers to read the text out loud to them. This requires a special kind of text within the site for the screen readers to access. If you would like to learn more about web accessibility, follow this link here. The World Wide Web Consortium (W3C) is responsible for setting web standards. The WCAG 2.0 standard was devised by the W3C as a guideline for making sites accessible to those with disabilities.
